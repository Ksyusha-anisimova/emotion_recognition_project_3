# Система распознавания эмоций на основе глубокого обучения

## Описание проекта
Дипломный проект по созданию системы распознавания эмоций человека по фотографии с использованием собственной реализации сверточной нейронной сети (CNN).
Текущая версия (B2) распознает 3 эмоции: **Счастье**, **Грусть**, **Нейтральное**.

## Особенности реализации
- Собственная архитектура CNN (не transfer learning)
- Реализация основных слоев нейросети с нуля для понимания внутренних процессов
- Веб-интерфейс для демонстрации работы системы
- Визуализация результатов и метрик обучения

## Технологический стек
- Python 3.8+
- PyTorch (для автоматического дифференцирования)
- Flask (веб-фреймворк)
- OpenCV (обработка изображений)
- NumPy, Matplotlib

## Структура проекта
```
emotion_recognition_project/
├── model/
│   ├── cnn_architecture.py    # Архитектура сети
│   ├── data_loader.py         # Загрузка и подготовка данных (legacy 7-классовый режим)
│   ├── train_fer2013_b2.py    # Обучение B2 (3 класса)
│   └── quick_train_b2.py      # Быстрое обучение на custom_train_b2
├── web_app/
│   ├── app.py                 # Flask приложение
│   ├── static/                # CSS, JS, изображения
│   └── templates/             # HTML шаблоны
├── utils/
│   └── visualization.py       # Визуализация результатов
└── requirements.txt
```

## Установка
```bash
pip install -r requirements.txt
```

## Запуск приложения
