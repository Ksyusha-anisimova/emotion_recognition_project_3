## СЛАЙД 4: Теоретические основы

### Сверточная нейронная сеть (CNN)

**Основные компоненты:**

```
Сверточный слой → Извлечение признаков
Batch Normalization → Стабилизация
Pooling → Снижение размерности
Dropout → Регуляризация
```

**Что извлекает CNN:**
- Слой 1-2: Края, линии
- Слой 3-4: Формы, паттерны
- Слой 5+: Сложные признаки

## СЛАЙД 5: Архитектура модели

### Структура сети

```
Input (48×48×1)
    ↓
[Conv-BN-ReLU-Conv-BN-ReLU-Pool-Dropout] × 4 блока
    ↓
Flatten → 4608 нейронов
    ↓
FC(512) → BN → ReLU → Dropout
    ↓
FC(256) → BN → ReLU → Dropout
    ↓
FC(3) → Softmax
    ↓
Output: 3 эмоции
```

**Параметры:** ~2.5 млн обучаемых весов

   - Haar Cascade Classifier
   - Обрезка по ROI

## СЛАЙД 8: Процесс обучения

### Параметры

- **Оптимизатор:** Adam (lr=0.0005)
- **Loss:** CrossEntropyLoss
- **Batch size:** 64
- **Эпохи:** 30
- **Регуляризация:**
  - Dropout: 0.25 / 0.5
  - Batch Normalization
  - Data Augmentation

### Scheduler
ReduceLROnPlateau (patience=5, factor=0.5)
