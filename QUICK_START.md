# 🚀 БЫСТРЫЙ СТАРТ

## Шаг 1: Установка зависимостей

```bash
# Клонирование или переход в директорию проекта
cd emotion_recognition_project

# Установка зависимостей
pip install -r requirements.txt
```

## Шаг 2: Запуск приложения

### Вариант A: Быстрая демонстрация (без обучения)

```bash
python run.py demo
```

Эта команда:
1. Проверит зависимости
2. Создаст демонстрационную модель
3. Запустит веб-приложение

Откройте браузер: **http://localhost:5000**

### Вариант B: Полное обучение модели (B2, 3 класса)

```bash
# Обучение модели (10 эпох по умолчанию)
python run.py train --epochs 30 --train-dir data/custom_train_b2

# Запуск приложения
python run.py run
```

### Вариант C: Ручной запуск

```bash
# Обучение (B2)
cd model
python train_fer2013_b2.py --train_dir ../data/custom_train_b2 --epochs 30

# Запуск веб-приложения
cd ../web_app
python app.py
```

## Требования к системе

- **Python**: 3.8+
- **RAM**: минимум 4 GB

- **Место на диске**: ~500 MB

## Возможные проблемы

### Ошибка: "No module named 'torch'"

```bash
pip install torch torchvision
```

### Ошибка: "No module named 'cv2'"

```bash
pip install opencv-python
```

### Порт 5000 занят

Измените порт в `web_app/app.py`:

```python
app.run(debug=True, host='0.0.0.0', port=8000)
```

### Веб-камера не работает

- Убедитесь, что браузер имеет разрешение на доступ к камере
- Используйте HTTPS или localhost
- Проверьте, что камера не используется другим приложением
